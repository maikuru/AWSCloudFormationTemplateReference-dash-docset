Amazon EMR Cluster JobFlowInstancesConfig ScalingTrigger

ScalingTrigger is a subproperty of the Amazon EMR Cluster JobFlowInstancesConfig property that specifies the conditions that trigger an automatic scaling activity.

Syntax

JSON

Copy
{
  "CloudWatchAlarmDefinition" : CloudWatchAlarmDefinition
}

YAML

Copy

  CloudWatchAlarmDefinition: CloudWatchAlarmDefinition

Properties

CloudWatchAlarmDefinition

The definition of a CloudWatch metric alarm. When the defined alarm conditions are met along with other trigger parameters, scaling activity begins.

Required: Yes

Type: Amazon EMR Cluster JobFlowInstancesConfig scalingtrigger
