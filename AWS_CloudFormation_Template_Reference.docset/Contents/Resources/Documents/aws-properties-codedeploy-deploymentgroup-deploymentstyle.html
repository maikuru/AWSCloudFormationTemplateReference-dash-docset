AWS CodeDeploy DeploymentGroup DeploymentStyle

The DeploymentStyle property type specifies attributes that determine the type of AWS CodeDeploy deployment, including specifying whether to route deployment traffic behind a load balancer for an AWS CodeDeploy deployment group.

Note

Currently, AWS CloudFormation only supports IN_PLACE deployment types.

DeploymentStyle is a property of the AWS::CodeDeploy::DeploymentGroup resource.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

Copy
{
  "DeploymentOption" : String
}

YAML

Copy
DeploymentOption: String

Properties

DeploymentOption

Indicates whether to route deployment traffic behind a load balancer. For valid values, see DeploymentStyle in the AWS CodeDeploy API Reference.

Required: No

Type: String

Update requires: No interruption
